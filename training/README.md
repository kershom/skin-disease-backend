# EfficientNet Training Pipeline

This folder contains the scripts used to train the EfficientNetB0 model for DermaLens.

Contents:
- preprocess.py – Dataset preprocessing
- split_dataset.py – Train/validation/test split
- train_efficientnet.py – Model training
- confusion_matrix.py – Evaluation
- classification_report.txt – Performance metrics
- confusion_matrix.png – Confusion matrix visualization

Note: The dataset and trained model files are not included in this repository.
